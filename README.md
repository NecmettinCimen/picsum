### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/necmettincimen/picsum.git
   ```
   Goto Directory
   ```sh
   cd picsum
   ```
2. Run Project
   ```sh
   npm i
   ```
3. Goto
   ```HTML
   npm start
   ```

### Publish & Release

1. Clone the repo
   ```sh
   git clone https://github.com/necmettincimen/picsum.git
   ```
   Goto Directory
   ```sh
   cd picsum
   ```
2. Build Image
   ```sh
    npm build
   ```
